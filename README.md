IT Infrastructure Management System

This is a web-based admin panel project developed using PHP, MySQL,and HTML/CSS.

Features
✔ Admin Login  
✔ Add Asset  
✔ View All Assets  
✔ Edit Asset  
✔ Delete Asset  
✔ Generate Asset Report  
✔ Export Asset Data to Excel

Technology Used
✔ PHP  
✔ MySQL  
✔ Bootstrap / CSS  
✔ XAMPP (Apache + MySQL)
